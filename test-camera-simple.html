📋 Instructions if Camera Not Opening:

  1. Allow Camera Permission: Look for 🔒 lock icon in address bar → Click → Allow Camera
  2. Close Other Apps: Close Zoom, Teams, Skype if open
  3. Restart Browser: Close and reopen browser
  4. Check Windows: Settings → Privacy & Security → Camera (make sure enabled)
